# Repo style

Team-local conventions for **this workspace**. Commit these files with the repo.

Suggested layout:

```text
STYLE.md                              # short map + pointers
.stack/guidance/style/repo/
  README.md                           # this file
  backend.md                          # optional focused docs
  frontend.md
```

Keep each file small and grep-friendly. Link to org synthstyle for global
rules instead of copying wholesale.
